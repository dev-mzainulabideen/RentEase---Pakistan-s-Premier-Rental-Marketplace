# RentEase - Pakistan's Premier Rental Marketplace

A modern rental marketplace platform built for connecting renters with property owners across Pakistan.

## Overview

RentEase is a comprehensive rental marketplace that simplifies the process of finding and listing properties, vehicles, equipment, and more. Built with modern web technologies, it provides a seamless experience for both renters and owners.

## Quick Start

1. Clone the repository
2. Open `index.html` in your browser
3. Start browsing listings instantly

## Features

- 🏠 Property listings
- 🚗 Vehicle rentals
- 🔍 Advanced search & filters
- 📍 Location-based search
- ⭐ Reviews & ratings
- 💬 Messaging system
- 📱 Fully responsive design

## Project Structure

```
frontend-html/
├── index.html           # Main landing page
├── search.html          # Search & browse listings
├── listing-detail.html  # Individual listing pages
├── create-listing.html  # Create new listings
├── profile.html         # User profile
├── dashboard.html       # User dashboard
├── css/                 # Stylesheets
├── js/                  # JavaScript files
└── logos/               # Brand assets
```

## Pages

- **Home** - Landing page with featured listings
- **Search** - Browse all available listings
- **Categories** - Browse by property type
- **Listing Detail** - Full listing information
- **Create Listing** - Add new property/vehicle
- **Profile** - User account management
- **Dashboard** - Manage your listings & bookings

## Technologies

- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap 5
- Mobile-first responsive design
- Vanilla JavaScript (no framework dependencies)

Open `index.html` in any modern web browser.
