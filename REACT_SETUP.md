# React Setup - Quick Start Guide

## 🚀 Quick Demo (No Installation)

**Simply open `react-demo.html` in your browser!**

This page uses React via CDN and works immediately without any setup.

---

## 📦 Full React App Setup

### Step 1: Install Dependencies

```bash
cd react-app
npm install
```

### Step 2: Start Development Server

```bash
npm start
```

The app will open at `http://localhost:3001`

### Step 3: Build for Production

```bash
npm run build
```

---

## ✅ What's Included

1. **React Components:**
   - StatsCard - Display statistics
   - ActivityFeed - Show recent activities
   - DashboardWidget - Container component

2. **API Integration:**
   - Connects to existing Express.js backend
   - Fetches data from `/api/admin/stats`
